API:
The 'itemart', 'gumpart', and 'multiart' APIs return PNG images. Hue is an optional parameter. The 'cliloc' API returns a plaintext string. Input may be decimal (10) or hexadecimal (0xA).
http://http://stormy-spire-5535.herokuapp.com//api/v1/itemart/(itemid)[/hue]
http://http://stormy-spire-5535.herokuapp.com//api/v1/gumpart/(itemid)[/hue]
http://http://stormy-spire-5535.herokuapp.com//api/v1/multiart/(itemid)[/hue]
http://http://stormy-spire-5535.herokuapp.com//api/v1/cliloc/(number)

Examples:
http://http://stormy-spire-5535.herokuapp.com//api/v1/itemart/7939/0x489
http://http://stormy-spire-5535.herokuapp.com//api/v1/gumpart/60469/1152
http://http://stormy-spire-5535.herokuapp.com//api/v1/multiart/55
http://http://stormy-spire-5535.herokuapp.com//api/v1/cliloc/500015
